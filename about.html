This is the implementation of my data visualisation project for the Data Visualisation (IN4089) course at the Delft University of Technology. It leverages the dataset from the CORGIS dataset project by Sam Donald containing data for different coffee beans.
The dataset contains information about different taste features of different coffee beans. This project tries to portray various insights and allow the user to observe correlations between the taste and the geographical location/processing method and other features.
